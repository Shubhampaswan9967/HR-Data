# HR Analytics Dashboard

## Project Overview
This project demonstrates an end-to-end HR Analytics Dashboard using Python, SQL, MySQL, Power Query, and Power BI. 
The HR dataset was downloaded from Kaggle, cleaned and preprocessed using Python (Pandas) in Jupyter Notebook, loaded into a MySQL database using Python, analyzed with SQL in MySQL Workbench, imported into Power BI from the MySQL database, transformed using Power Query, and visualized through an interactive dashboard to generate HR insights.

## Tools & Technologies
- Python
- Pandas
- SQL
- MySQL Workbench
- Power Query
- Power BI

## Project Workflow
1. Downloaded the HR dataset from kaggle.
2. Cleaned and preprocessed the data using Python (Pandas) in Jupyter Notebook.
3. Loaded the cleaned data into a MySQL database using python.
4. Performed data analysis using SQL in MySQL Workbench.
5. Imported data from the MySQL database into Power BI.
6. Transformed and prepared the data using Power Query.
7. Built an interactive HR Analytics Dashboard in Power BI.

## Key Performance Indicators (KPIs)
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company

## Dashboard Features
- Attrition by Job Role
- Attrition by Overtime
- Attrition by Department
- Attrition by Gender
- Average Salary by Department
- Average Monthly Income by Job Role
- Job Satisfaction Distribution
